Stella Home

Application mobile développée pour gérer les réservations d’une maison familiale à Stella Plage, avec synchronisation en temps réel via Firebase.

Rôle — Développement mobile complet

Stack — Flutter · Firebase · Firestore · iOS

← Retour aux projets

Overview

Suite à l’achat d’une maison familiale à Stella Plage, j’ai développé une application mobile permettant de gérer les réservations et d’éviter les conflits de dates. L’objectif était de proposer un outil simple, accessible à toute la famille, synchronisé en temps réel et disponible directement sur iPhone.

What I did

1. Développement mobile avec Flutter

L’application a été développée en Flutter afin de construire une interface mobile fluide et moderne. Les principales fonctionnalités incluent la consultation du calendrier, la création de réservations et la visualisation des séjours à venir.

2. Mise en place de la base de données Firestore

Une base de données NoSQL a été hébergée sur Firebase / Firestore. La structure repose sur une collection de réservations, permettant un stockage flexible et une synchronisation en temps réel entre les différents utilisateurs de l’application.

3. Connexion front mobile ↔ base de données

L’application est connectée directement à Firestore. Chaque réservation créée ou modifiée est instantanément mise à jour pour tous les utilisateurs. Cette synchronisation garantit la cohérence des données et évite les conflits de planning.

4. Compilation et déploiement iOS

Une fois l’application finalisée, elle a été compilée et exportée en application iOS. Cela m’a permis de gérer les certificats, la configuration Xcode et le packaging mobile jusqu’à une version utilisable directement sur iPhone.

Tech & tools

Flutter pour le développement mobile cross-platform, Dart pour la logique applicative, Firebase Authentication pour la gestion des utilisateurs, Cloud Firestore comme base de données NoSQL en temps réel, et Xcode pour la compilation et le déploiement iOS.

What I learned

Ce projet m’a permis de développer une application mobile complète, depuis l’interface jusqu’au stockage des données et au déploiement. J’ai appris à structurer une base NoSQL adaptée à un cas d’usage réel, à gérer la synchronisation en temps réel et à mener un projet jusqu’à une application installable sur iOS.
